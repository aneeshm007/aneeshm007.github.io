Introduction to the Brusselator

Alan Turing's 1952 paper The Chemical Basis for Morphogenesis describes the origins of skin pigmentations due to unstable reactions of chemical substances called "morphogens", which diffuse along a concentration gradient in skin and react together. Turing modeled these reactions and found they produce patterns commonly found in nature, such as the spots and stripes on animals. Here we model these unstable reaction-diffusion equations in the Brusselator model on the grid \([0,40] x [0,40]\) in order to see these patterns.
This model is given by the following equations using homogeneous Neumann boundary conditions. \begin{align} p_t = D_p\nabla^2 p + p^2q + C - (K+1)p \\ q_t = D_p\nabla^2 q - p^2q + Kp\\ p(x,y,0) = C + 0.1 \\ q(x,y,0) = \frac{K}{C} + 0.2 \end{align}

We expect to see turing instability when $$ K > (1 + C\sqrt{\frac{D_p}{D_q}})^2$$

Veryifying Equilibrium Solutions

Next we will prove that \(p = c\) and \(q = \frac{K}{C}\) are equilibrium solutions for the equations described in the introduction. We note that \(\nabla^2 p = \nabla^2 q = 0\) since \(p\) and \(q\) are constants. Therefore, we will show \begin{align} 0 = p^2q + C - (K+1)p\\ 0 = -p^2q + Kp \\ \end{align} If we substitute \(p = c\) and \(q = \frac{K}{C}\), we obtain \begin{align} 0 = C^2(\frac{K}{C}) + C - (K+1) C \\ 0 = -C^2*(\frac{K}{C}) + KC \\ \end{align} We see that $$ CK + C - KC - C = 0$$ and $$ CK + KC = 0$$. Therefore we conclude \(p = c\) and \(q = \frac{K}{C}\) are equilibrium solutions.
